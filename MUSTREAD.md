This file outlines the core academic papers that support our project's proposed methodology. These papers provide the precedent for our proposed architecture, including the use of a two-stage pipeline (**YOLO + CNN**) and **transfer learning**.

## 1. Core Paper on Model Comparison

**Title:** *Detecting Emotions from Facial Features Using Deep Learning and Computer Vision* (2025)  

**Why it's important:**  
This paper directly compares the performance of **YOLOv5**, **YOLOv8**, and **VGG16** for emotion detection, which is highly relevant to validating our proposed tech stack.

---

## 2. Paper on Transfer Learning for Stage 2

**Title:** *Automatic Facial Recognition Using VGG16 Based Transfer Learning Model* (2020)  

**Why it's important:**  
This paper details the use of **VGG16** with transfer learning on the **CK+** and **JAFFE** datasets. It provides a strong precedent for building our Stage 2 classifier.

---

## 3. Paper on Real-Time Implementation

**Title:** *Facial Emotion Recognition Based Real-Time Learner Engagement Detection System in Online Learning Context Using Deep Learning Models* (2022)  
**Authors:** S. Gupta, P. Kumar, and R.K. Tekchandani  
**Publication:** *Multimedia Tools and Applications*  

**Why it's important:**  
This paper discusses the implementation of a real-time system using models like **Inception-V3** and **ResNet-50**. It provides valuable insights into the challenges of processing a live video feed.
